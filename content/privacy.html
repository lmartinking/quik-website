Privacy Policy

Quik: Gravity Flip Platformer, by Irrational Idiom

Personal Information

No personal information is collected, used or stored.

High Scores

The game software may connect to scores.irrationalidiom.com to retrieve and submit high scores.

The following information is transmitted by the game software, when the game is submitting a high score to scores.irrationalidiom.com

  • 3 character nick name (eg: AAA, ABC, DDD)
  • user high score
  • game user agent (eg: Quik/1.0.0)

The 3 character nick names and high scores are stored by the high score server software running on scores.irrationalidiom.com.
